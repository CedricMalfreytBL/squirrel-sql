Overview

SQuirreL SQL Client is a graphical Java program that will allow you to view the structure of a JDBC compliant database, browse the data in tables, issue SQL commands etc.

It was originally released under the GNU General Public License. Since version 1.1beta2 it has been released under the GNU Lesser General Public License. The change log gives the history so far.

If you have any queries or issues about SQuirreL, please post them to the Users Mailing list.

  

 

Requirements

SQuirreL SQL Client 1.0 requires JDK1.2 to work but prefers JDK1.3.

SQuirreL SQL Client 1.1 and above require JDK1.3.

 

 

Installation

The SQuirreL SQL Client is installed with the IzPack Java installer. Download the file squirrel-sql-<version>-install.jar and execute it using the following command:

java -jar squirrel-sql-<version>-install.jar

Note that some browsers may attempt to open this file rather than download it. In this case, right-click on the download link and click your browser's equivalent of the "Save as..." menu item.

On Windows, execute the file squirrel-sql.bat to run the application. On Unix, the file is squirrel-sql.sh.

To install plug-ins for SQuirreL SQL Client 1.1 and above, you need to download the appropriate .zip or .tar.gz file and unpack it into the plugins directory within the SQuirreL SQL Client application directory. Please be sure that you select the "use folder names" option. Source for each plug-in is also supplied within the file.
