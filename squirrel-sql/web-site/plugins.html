Overview

A plugin is an application written in Java that runs within SQuirreL. This allows developers to enhance the functionality of SQuirreL without having to rebuild the SQuirreL application itself.

  

 

Installation

To install plugins for SQuirreL SQL Client 1.1 and above, you need to download the appropriate .zip or .tar.gz file from the Downloads page and unpack it into the plugins directory within the SQuirreL SQL Client application directory.

Please be sure that you select the "use folder names" option. The source for each plugin is supplied with it.

  

 

Existing Plugins

jEdit by Colin Bell and Johan Compagner
Version 0.16 released 26 August 2002

Requirements: SQuirreL SQL Client 1.1beta6+ and JDK1.3+

This plugin replaces the standard Java text area control with one from the jEdit text editor. Additional features include syntax highlighting, bracket matching and much more.

Look and Feel by Colin Bell
Version 0.22 released 26 August 2002

Requirements: SQuirreL SQL Client 1.1beta6+ and JDK1.3+

This plugin allows you to select a Look and Feel other than the default Java one. It also allows you to specify the fonts for the different GUI elements. It ships with several Look and Feel implementations.

Session Scripts by Colin Bell
Version 0.11 released 26 August 2002

Requirements: SQuirreL SQL Client 1.1beta6+ and JDK1.3+

The Sesssion Scripts plugin allows you to specify one or more SQL statements to be executed when a session is started.

SQL Scripts by Johan Compagner
Version 0.20 released 26 August 2002

Requirements: SQuirreL SQL Client 1.1beta6+ and JDK1.3+

The SQL Scripts plugin allows you to save and restore SQL scripts from a text file. It will also allow you to generate CREATE TABLE and INSERT scripts for existing tables in your database.
 

 

Plugins under development

Data Import by Like Gao

The Data Import plugin will allow you to import data into existing and new tables in a variety of formats. The first format will be CSV.
